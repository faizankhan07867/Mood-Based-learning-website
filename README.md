# Mood-Based-learning-website


🚀 Mood Based Learning Platform – Project Roadmap
Phase 1 — Project Idea & Problem

Goal: Improve student learning by considering their emotional state.

Problem:

Students often feel tired, stressed, or motivated

Traditional learning platforms ignore mood and focus level

Solution:

A platform that recommends learning content based on mood
